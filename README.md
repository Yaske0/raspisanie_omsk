# ✈ Расписание смены — Омский аэропорт

Программа «в одну кнопку»: сама определяет вашу рабочую смену, скачивает
онлайн-табло Омского аэропорта (прилёт и вылет с учётом задержек), формирует
документ Word по нужному образцу и отправляет его на печать.

Нажал кнопку → собрано расписание → готов документ → ушёл на принтер.

![Интерфейс программы](docs/preview.png)

---

## Зачем это

Раньше расписание на смену приходилось собирать вручную: смотреть табло,
выбирать рейсы, которые попадают в рабочие часы, отдельно учитывать задержанные
рейсы и перепечатывать всё в Word. Программа делает это за пару секунд и одним
нажатием.

## Возможности

- **Одна кнопка** — весь процесс (сбор → документ → печать) автоматический.
- **Понимает смены.** График цикличный (4 дня: день → ночь → выходной →
  выходной); программа сама вычисляет текущую или ближайшую смену.
  - Дневная смена 07:45–19:45, ночная 19:45–07:45 следующего дня.
- **Учитывает задержки.** Рейс, который по расписанию должен был быть раньше, но
  задержан в вашу смену, — попадёт в список. Рейс, уехавший за конец смены, —
  убирается. Если известно новое время — оно подставляется; если нет — пишется
  **ЗАДЕРЖАН**.
- **Документ Word по образцу** — шрифт Verdana (заголовки 28, текст 11), две
  таблицы «Прилет» и «Вылет», всё умещается на одну страницу.
- **Печать** на принтер по умолчанию без лишних окон.
- **Современный интерфейс** — тёмно-синяя авиационная тема, поддержка масштаба
  Windows (100/125/150 %), анимация во время работы.
- **Не требует установки** — готовый `.exe`, ничего ставить не нужно.

## Как это работает

1. **Смена.** От «якорной» дневной смены (по умолчанию 25.06.2026) считается
   4-дневный цикл, и определяется окно текущей/ближайшей смены.
2. **Сбор данных.** Программа запрашивает онлайн-табло аэропорта (прилёт и вылет,
   при необходимости — сегодня и завтра для ночной смены).
3. **Отбор рейсов.** В список попадают рейсы, чьё фактическое/ожидаемое время —
   внутри окна смены, с учётом задержек.
4. **Документ.** Данные раскладываются в таблицы Word по образцу.
5. **Печать.** Готовый файл уходит на принтер по умолчанию.

Источник данных — публичное онлайн-табло
[Омского аэропорта](https://www.aeroomsk.ru/passazhiram/onlayn-tablo/).

## Установка (получить .exe без установки чего-либо)

Сборка делается бесплатно в облаке GitHub Actions — на компьютере ничего ставить
не нужно, только браузер.

1. Загрузите файлы проекта в репозиторий (вкладка **Add file → Upload files**),
   включая папки `aeroomsk`, `.github` и `docs`.
2. Откройте вкладку **Actions** — сборка запустится сама (или нажмите
   **Run workflow**). Идёт 3–5 минут.
3. В завершённом запуске внизу, в разделе **Artifacts**, скачайте
   **РасписаниеСмены-windows**.
4. Распакуйте архив. Внутри — `РасписаниеСмены.exe` и `config.json`. Положите их
   в одну папку.

Дальше каждый раз: двойной клик по `РасписаниеСмены.exe` → кнопка
**«Создать и напечатать»**.

> Подробная пошаговая инструкция со скриншотами — в [docs/BUILD.md](docs/BUILD.md).

### Запуск из исходников (для разработчиков)

```bash
pip install -r requirements.txt
python run.py
```

## Настройка — `config.json`

Лежит рядом с программой, открывается Блокнотом. Пересобирать ничего не нужно.

```json
{
    "anchor_day_shift": "2026-06-25",
    "printer_name": "",
    "auto_print": true,
    "output_dir": ""
}
```

| Параметр | Что задаёт |
|----------|------------|
| `anchor_day_shift` | дата, когда точно была **дневная** смена (от неё считается цикл) |
| `printer_name` | имя принтера; пусто = принтер по умолчанию |
| `auto_print` | `true` — печатать сразу, `false` — только создавать файл |
| `output_dir` | куда складывать документы; пусто = папка «Расписания» рядом с программой |

Хотите свою анимацию вместо динозаврика — положите рядом с `.exe` файл
`loading.gif`, программа возьмёт его автоматически.

## Состав проекта

| Файл | Назначение |
|------|------------|
| `aeroomsk/shifts.py` | расчёт смены по 4-дневному циклу |
| `aeroomsk/tablo.py` | загрузка и разбор онлайн-табло |
| `aeroomsk/builder.py` | отбор рейсов под смену и правила задержек |
| `aeroomsk/docxgen.py` | сборка документа Word по образцу |
| `aeroomsk/printing.py` | печать на Windows |
| `aeroomsk/core.py` | связывает всё вместе |
| `aeroomsk/app.py` | окно с кнопкой (интерфейс) |
| `.github/workflows/` | автоматическая сборка `.exe` в облаке |

## Оговорки

Это неофициальная вспомогательная программа. Данные берутся с публичного
онлайн-табло аэропорта «как есть»; при крупном изменении сайта может
потребоваться правка модуля разбора. Расписание носит справочный характер.
